# 📱 GUIDA COMPLETA — MatchBet Pro su Android
# Come trasformare l'app in APK installabile

=======================================================
COSA TI SERVE:
- Telefono o PC con browser (Chrome va benissimo)
- Email (per creare account GitHub)
- 15-20 minuti la prima volta
=======================================================

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 1 — CREA ACCOUNT GITHUB (2 minuti)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Vai su: https://github.com
2. Clicca "Sign up" (in alto a destra)
3. Inserisci email, password, username (es: matchbet2024)
4. Verifica l'email
5. Scegli il piano GRATUITO (Free)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 2 — CREA UN REPOSITORY (1 minuto)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Dopo il login, clicca il "+" in alto a destra
2. Seleziona "New repository"
3. Compila così:
   - Repository name: matchbet-pro
   - Description: MatchBet Pro App
   - Seleziona: Public (IMPORTANTE - serve per Actions gratis)
   - Spunta: "Add a README file"
4. Clicca "Create repository"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 3 — CARICA I FILE (5 minuti)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Devi caricare questi file nella cartella del repository:

FILE DA CARICARE (li hai già da Claude):
  - matchbet_pro_v9.html  →  rinominalo "index.html"
  - capacitor.config.json
  - package.json
  - build-apk.yml  (questo va in una sottocartella speciale)

COME CARICARE:

A) Carica prima i file principali:
   1. Nella pagina del repository, clicca "Add file" → "Upload files"
   2. Trascina (o seleziona) questi 3 file:
      - index.html
      - capacitor.config.json  
      - package.json
   3. In fondo scrivi nel campo "Commit": "Add app files"
   4. Clicca "Commit changes"

B) Crea la cartella .github/workflows e carica il workflow:
   1. Clicca "Add file" → "Create new file"
   2. Nel campo nome file scrivi esattamente:
      .github/workflows/build-apk.yml
      (GitHub crea le cartelle automaticamente)
   3. Copia e incolla il contenuto del file build-apk.yml
   4. Clicca "Commit new file"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 4 — AVVIA LA BUILD (automatica!)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Dopo aver caricato il file build-apk.yml, GitHub parte
automaticamente con la compilazione.

Per vedere lo stato:
1. Clicca sulla scheda "Actions" (in alto nel repository)
2. Vedrai "Build MatchBet Pro APK" in esecuzione
3. L'icona gialla = in corso (dura 5-10 minuti)
4. L'icona verde = COMPLETATO ✅
5. L'icona rossa = errore (rarissimo, dimmi il messaggio)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 5 — SCARICA L'APK
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Clicca sulla build completata (icona verde)
2. Scorri in basso fino ad "Artifacts"
3. Clicca "MatchBet-Pro-APK" per scaricare
4. Estrai lo zip → trovi "app-debug.apk"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 6 — INSTALLA SUL TELEFONO ANDROID
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Prima di installare, abilita "Fonti sconosciute":
(da fare UNA SOLA VOLTA)

Android 8+:
  Impostazioni → App → (nome browser) → Installa app sconosciute → Attiva

Android 7 o precedenti:
  Impostazioni → Sicurezza → Origini sconosciute → Attiva

Poi:
1. Trasferisci app-debug.apk sul telefono
   (via email, WhatsApp, Google Drive, cavo USB...)
2. Apri il file sul telefono
3. Tocca "Installa"
4. Tocca "Apri"

L'app appare nella lista app con l'icona MatchBet Pro!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
COME AGGIORNARE L'APP IN FUTURO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Quando Claude aggiorna il file HTML:
1. Vai sul repository GitHub
2. Clicca su "index.html" → matita (modifica) → incolla il nuovo contenuto
   OPPURE cancella il vecchio e carica il nuovo index.html
3. GitHub ricompila automaticamente
4. Scarica il nuovo APK e reinstalla

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
CONDIVIDERE CON AMICI
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Per condividere l'APK con i tuoi amici:
- Scarica l'APK e mandalo via WhatsApp/Telegram/Drive
- Loro devono solo abilitare "Fonti sconosciute" e installarlo
- GRATIS, nessun account necessario per loro

=======================================================
PROBLEMI COMUNI:
=======================================================

❌ "Non riesco a installare l'APK"
→ Assicurati di aver abilitato "Fonti sconosciute"

❌ "La build fallisce (icona rossa)"
→ Manda screenshot dell'errore a Claude

❌ "Non trovo il file .github/workflows"
→ Il punto davanti a .github è importante! Non dimenticarlo

❌ "Actions non parte"
→ Vai su Actions → Enable Actions → conferma
